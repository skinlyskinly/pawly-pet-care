# 🐾 Pawly — AI 반려동물 돌봄 앱

AI 학습 기반 **맞춤 관리**와 **감정 인식**을 결합한 반려동물 돌봄 웹앱입니다.
보호자가 외출·업무 중에도 반려동물의 건강·감정·활동량을 한눈에 확인하고,
AI가 추천하는 맞춤 케어를 받을 수 있습니다.

순수 **HTML / CSS / JavaScript**로만 작성되어 별도 빌드 없이 **GitHub Pages**에 바로 배포됩니다.

## ✨ 핵심 기능

| 기능 | 설명 |
|------|------|
| 🏠 **홈 대시보드** | 현재 감정, 건강 점수, 활동량, 수면, 주간 활동 차트, 오늘의 AI 추천 |
| 💞 **감정 분석** | AI가 분석한 감정 분포 · 하루 감정 타임라인 · 학습한 행동 패턴 |
| 🐾 **나비 상태** | 감정·건강 상태에 따라 **고양이 모습이 실시간으로 변하는 애니메이션** |
| 🤖 **AI 케어** | 품종·체중·감정 기반 맞춤 케어 플랜과 돌봄 체크리스트 |

## 🐾 상태 애니메이션 (SVG + CSS)

순수 SVG와 CSS 애니메이션으로 그린 고양이가 감정·건강 상태에 따라
표정·귀·꼬리·동작과 효과를 바꿔 보호자가 한눈에 상태를 알 수 있습니다.

| 상태 | 표현 |
|------|------|
| 😊 편안해요 | 잔잔한 호흡 · 느린 눈깜빡임 · 미소 |
| 😻 행복해요 | 웃는 눈 · 하트 · 빠른 꼬리 흔들기 |
| 😴 졸려요 | 감은 눈 · 떠오르는 Zzz · 느린 호흡 |
| 😼 경계 중 | 커진 동공 · 쫑긋 선 귀 · 몸 떨림 · 느낌표 |
| 🥺 외로워요 | 처진 눈썹 · 눈물 · 축 처진 자세 |
| 🤕 아파요 | 창백한 털 · 얼음주머니 · 식은땀 · 수의사 상담 알림 |

> 기본 상태는 AI가 분석한 감정·건강 값을 반영하며, 화면에서 상태를 눌러 미리 볼 수도 있습니다.
> `prefers-reduced-motion` 환경에서는 애니메이션이 자동으로 멈춥니다.

## 📁 구조

```
.
├── index.html        # 앱 마크업
├── css/styles.css    # Figma UI3 스타일 (화이트 + 블루·민트)
├── js/data.js        # 반려동물 · AI · 상태 목업 데이터
├── js/app.js         # 대시보드 렌더링 · 상태 애니메이션 로직
└── .claude/          # 로컬 미리보기용 (배포에 영향 없음)
```

## 🚀 로컬 실행

간단한 정적 서버로 열면 됩니다.

```bash
# Python
python3 -m http.server 8000
# 또는 Node
npx serve .
```
브라우저에서 `http://localhost:8000` 접속.

## 🌐 GitHub Pages 배포

1. 이 폴더를 새 GitHub 저장소에 푸시
2. **Settings → Pages → Source**를 `main` 브랜치 `/ (root)`로 설정
3. 발급된 `https://<사용자명>.github.io/<저장소>/` 주소로 접속

빌드 단계가 없어 푸시만 하면 바로 게시됩니다.

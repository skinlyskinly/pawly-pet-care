# 🐾 Pawly — AI 반려동물 건강 체크

반려동물의 일상 컨디션을 **기록**하고, **카메라 기반 AI 분석**으로 변화를 살펴볼 수 있는 웹앱입니다.
집에서 "오늘은 평소와 좀 다른가?" 싶을 때, 식사·물·활동·배변·기분을 가볍게 체크하고
카메라로 아이의 얼굴·자세·움직임을 확인할 수 있습니다.

순수 **HTML / CSS / JavaScript**로만 작성되어 별도 빌드 없이 **GitHub Pages**에 바로 배포됩니다.

## ✨ 기능

| 기능 | 설명 |
|------|------|
| 🏠 **홈** | 오늘 컨디션 한눈에 보기 · 항목별 요약 · 최근 7일 추이 |
| 📝 **오늘 체크** | 식사량·물·활동량·배변·기분을 빠르게 기록 (+메모) |
| 📷 **AI 분석** | 카메라로 반려동물을 감지하고 움직임·자세를 분석 |
| 📅 **기록** | 날짜별 건강 기록 저장·열람·삭제 |
| 🐶 **프로필** | 이름·나이·종·특징 등록 |

## 🧠 컨디션 판정

각 체크 항목의 비정상 신호를 모아 보호자가 이해하기 쉬운 문장으로 보여줍니다.

- **평소와 비슷함** — 전반적으로 안정적
- **주의 필요** — 식사 거부·무기력·묽은 변 등 변화가 겹칠 때
- **기록 권장** — 아직 기록이 없거나 카메라로 잘 확인되지 않을 때

## 📷 카메라 AI 분석 (mediapipe/tasks-vision)

[`@mediapipe/tasks-vision`](https://www.npmjs.com/package/@mediapipe/tasks-vision)의
**ObjectDetector**(EfficientDet-Lite)를 CDN에서 동적 로드해 사용합니다.

1. 영상 프레임에서 반려동물(고양이·강아지·새 등)을 감지
2. 바운딩 박스의 **중심 이동량**으로 움직임 정도를, **가로/세로 비율**로 자세를 추정
3. 감지율·움직임·자세를 종합해 "평소와 비슷함 / 주의 필요 / 기록 권장" 문장으로 안내

> 분석은 모두 **브라우저(기기) 안에서** 처리되며 영상은 저장·전송되지 않습니다.
> 결과는 버튼 한 번으로 오늘 기록에 함께 남길 수 있습니다.

## 💾 데이터 저장

모든 기록·프로필은 브라우저 **localStorage**에 저장됩니다. (`pawly:records`, `pawly:profile`)

## 📁 구조

```
.
├── index.html     # 마크업 (홈·체크·AI분석·기록·프로필)
├── style.css      # Figma UI3 스타일 (베이지/아이보리/소프트 그린)
├── script.js      # 상태 판정 · 기록 · 카메라 AI 로직 (ES Module)
└── .claude/       # 로컬 미리보기용 (배포에 영향 없음)
```

## 🚀 로컬 실행

ES Module과 카메라(getUserMedia)를 쓰므로 정적 서버로 열어야 합니다.

```bash
python3 -m http.server 8000
# 또는
npx serve .
```
브라우저에서 `http://localhost:8000` 접속. (카메라는 `localhost` 또는 `https`에서만 동작)

## 🌐 GitHub Pages 배포

1. 이 폴더를 GitHub 저장소에 푸시
2. **Settings → Pages → Source**를 `main` 브랜치 `/ (root)`로 설정
3. 발급된 `https://<사용자명>.github.io/<저장소>/` 로 접속 (HTTPS라 카메라도 동작)

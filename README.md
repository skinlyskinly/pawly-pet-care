# 🐾 Pawly — AI 반려동물 돌봄 앱

AI 학습 기반 **맞춤 관리**와 **감정 인식**을 결합한 반려동물 돌봄 웹앱입니다.
보호자가 외출·업무 중에도 반려동물의 건강·감정·활동량을 한눈에 확인하고,
AI가 추천하는 맞춤 케어를 받을 수 있습니다.

순수 **HTML / CSS / JavaScript**로만 작성되어 별도 빌드 없이 **GitHub Pages**에 바로 배포됩니다.

## ✨ 핵심 기능

| 기능 | 설명 |
|------|------|
| 🏠 **홈 대시보드** | 현재 감정, 건강 점수, 활동량, 수면, 주간 활동 차트, 오늘의 AI 추천 |
| 💞 **감정 분석** | AI가 분석한 감정 분포 · 하루 감정 타임라인 · 학습한 행동 패턴 |
| ✋ **제스처 교감** | 카메라로 보호자의 **손동작을 인식**하고 반려동물이 반응 (MediaPipe) |
| 🤖 **AI 케어** | 품종·체중·감정 기반 맞춤 케어 플랜과 돌봄 체크리스트 |

## ✋ 손동작 인식 (MediaPipe Tasks Vision)

`@mediapipe/tasks-vision`의 `GestureRecognizer`로 손동작을 실시간 인식하여
반려동물 케어 동작으로 매핑합니다.

| 손동작 | 의미 | 반려동물 반응 |
|--------|------|----------------|
| 👍 따봉 | 칭찬 | 골골송을 부르며 비벼요 |
| ✋ 손바닥 | 이리와 | 꼬리를 세우고 다가와요 |
| ✌️ 브이 | 놀이 | 사냥 자세를 잡아요 |
| ✊ 주먹 | 기다려 | 식빵 자세로 대기 |
| ☝️ 검지 | 주목 | 손끝을 응시 |
| 🤟 사랑해 | 애정 | 천천히 눈을 깜빡여요(눈인사) |

> 카메라 권한이 필요하며, **HTTPS(또는 localhost)** 환경에서만 작동합니다.
> GitHub Pages는 HTTPS로 제공되므로 그대로 동작합니다.

## 📁 구조

```
.
├── index.html        # 앱 마크업
├── css/styles.css    # Figma UI3 스타일 (화이트 + 블루·민트)
├── js/data.js        # 반려동물 · AI 목업 데이터
├── js/gesture.js     # MediaPipe 손동작 인식 모듈
├── js/app.js         # 대시보드 렌더링 · 제스처 교감 로직
└── .claude/          # 로컬 미리보기용 (배포에 영향 없음)
```

## 🚀 로컬 실행

ES 모듈을 사용하므로 `file://`이 아닌 로컬 서버로 열어야 합니다.

```bash
# Python
python3 -m http.server 8000
# 또는 Node
npx serve .
```
브라우저에서 `http://localhost:8000` 접속.

## 🌐 GitHub Pages 배포

1. 이 폴더를 새 GitHub 저장소에 푸시
2. **Settings → Pages → Source**를 `main` 브랜치 `/ (root)`로 설정
3. 발급된 `https://<사용자명>.github.io/<저장소>/` 주소로 접속

빌드 단계가 없어 푸시만 하면 바로 게시됩니다.
